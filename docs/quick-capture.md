# 快捷记录模块设计

## 目标
- 提供独立的快捷记录弹窗，便于快速采集文本。
- 支持保存到今日日记或新建文件。
- 在锁屏状态下保持弹窗可输入、可保存，不影响其他窗口锁定。

## 核心能力
- 命令面板命令：`打开快捷记录窗口`。
- 支持插件内置 Electron 全局热键，绕开第三方 `Global Hotkeys` 插件。
- 弹窗打开后自动聚焦编辑区。
- 唤起快捷记录时不会强制激活主窗口。
- `Ctrl+S` / `Cmd+S` 保存后自动关闭弹窗。
- 同一时刻仅维护一个快捷记录会话，重复触发时激活已存在窗口。
- 默认窗口尺寸为 `700 x 300`。
- 弹窗聚焦会进行多次重试，并自动将光标定位到文末，便于立即续写。
- 若浏览器级 `focus()` 不稳定，会额外尝试 Electron 原生窗口聚焦，尽量避免主窗口被重新拉起。
- 对全局热键等场景做了延迟重试聚焦与源窗口失焦处理，降低主窗口回抢焦点概率。
- 当前策略下，速记窗口不会主动隐藏主窗口。
- 速记窗口保存关闭时会抑制已知的 `ResizeObserver` 收尾告警，避免控制台出现误导性报错。

## 保存策略
- `daily`：保存到今日日记文件。
  - 目录可配置（可为空）。
  - 文件名格式可配置（支持 `YYYY`、`MM`、`DD`）。
- `new-file`：每次新建独立文件。
  - 目录可配置。
  - 文件名前缀可配置。
  - 自动追加时间戳，冲突时附加随机后缀。

## 锁屏豁免集成
- 快捷记录弹窗打开后自动设置豁免标记：
  - `window.__LOCK_SCREEN_EXEMPT__ = true`
  - `document.documentElement[data-lockscreen-exempt="true"]`
- 锁屏插件会识别豁免标记，确保该弹窗不进入锁定遮罩。
- 关闭弹窗时自动移除豁免标记。
- 主窗口、副窗口继续按全局锁定策略工作，互不影响。

## 会话生命周期
1. 触发命令后延迟打开弹窗（避免命令面板焦点干扰）。
2. 创建或获取目标文件。
3. 创建弹出窗口并打开目标文件。
4. 绑定保存快捷键监听。
5. 检测到 `Ctrl+S/Cmd+S` 后等待文件保存事件。
6. 保存完成后自动关闭会话。

## 配置项
- `enabled`：启用快捷记录。
- `useGlobalShortcut`：是否启用内置全局热键。
- `globalShortcut`：Electron Accelerator 格式的全局热键。
- `saveTarget`：保存策略（`daily` / `new-file`）。
- `dailyFolder`：今日日记目录。
- `dailyFileNamePattern`：今日日记文件名格式。
- `newFileFolder`：新建文件目录。
- `newFileNamePrefix`：新建文件名前缀。

## 异常处理
- 目录路径冲突（同名文件）会中止并提示错误。
- 记录窗口创建失败时会自动回退 `getLeaf("window")`。
- 弹窗丢失会话时，会在窗口关闭事件中自动清理。
- 若全局热键被其他程序占用，注册会失败并给出提示。
- 快捷记录窗口打开时不再弹出保存提示，由用户直接输入并使用 `Ctrl+S/Cmd+S` 保存。
